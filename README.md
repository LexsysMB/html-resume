# Personal Resume Website

## Overview

This repository contains my personal resume website created as part of the **WGU D276 – Web Development Foundations** course. The project demonstrates fundamental web development skills by building a responsive resume using HTML and CSS.

## Features

- Responsive resume layout
- Semantic HTML5 structure
- Custom CSS styling
- Organized sections for education, skills, and experience
- Clean and simple user interface

## Technologies Used

- HTML5
- CSS3
- Git
- GitHub

## Learning Objectives

Through this project, I gained experience with:

- Structuring web pages using semantic HTML
- Styling webpages with CSS
- Creating responsive layouts
- Organizing project files
- Using Git and GitHub for version control
